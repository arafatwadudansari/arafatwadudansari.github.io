CMD Explorer

The purpose of this program is to provide a basic graphical interface for running common command-line tools like ping, tracert, and nslookup. It simplifies command execution by offering an interactive window.

  • Key features include the ability to run selected commands directly from the GUI, enter additional details when needed, and receive error alerts if no command is selected or if inputs are missing.
  • The interface is created using Tkinter, which handles the layout and manages pop-up alerts. The Subprocess library is used to execute the commands and process their output.

Check out the code on my GitHub project repository: https://github.com/ArafatWadudAnsari/CMD-Explorer

Network Performance Tool

The purpose of this program is to offer a basic graphical interface for monitoring basic network performance. It provides real-time updates on latency, local IP address, and the number of devices connected to the network.

  • Key features include checking latency to an external IP, retrieving the local IP address, and counting connected devices. Users can refresh the data with a single click and receive error alerts if there are issues.
  • The interface is built using Tkinter for layout and updates. The Subprocess library executes commands and processes network data, while Socket retrieves the local IP address, and re extracts and counts unique IP addresses from the command output.

Check out the code on my GitHub project repository: https://github.com/ArafatWadudAnsari/Network-Performance-Tool
